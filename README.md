# Community 🌐

A repo with some Graph Community Detection algorithms implemented from scratch. (and no AI)

Algorithms Currently Implemented:

- Louvain Algorithm
- Leiden Algorithm (TBC)

## Repo Set-up 🚀

This repo uses uv as the python package and project manager. To install uv, do:

```powershell
pip install uv
```

Thereafter, you can easily create your virtual environment by doing:

```powershell
uv sync --all-extras
```

To activate the virtual environment, do:

- Windows

```
.venv\scripts\activate
```

- MacOS

```
.venv\bin\activate
```
